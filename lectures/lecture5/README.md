## 🚀 ローカル起動方法

プロトタイプを手元で実行するには、以下のコマンドを実行します。

```bash
cd lectures/lecture5/prototype
npm install
npm run dev
```

Vite開発サーバーが起動したら、ブラウザで [http://localhost:5173/](http://localhost:5173/) にアクセスしてください。
